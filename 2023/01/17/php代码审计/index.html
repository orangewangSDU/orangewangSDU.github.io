bak是备份文件，为文件格式扩展名。
这类文件一般在.bak前面加上应该有原来的扩展名
那就访问备份文件！
直接在网址栏后面/index.php.bak
下载了该文件，打开是一段最简单的PHP代码
对于我这种啥也不会的新手已经非常非常非常非常非常友好了！！！

现在开始学总比老是拖着强！

  1. include_once() 语句是指在脚本执行期间包含并运行指定文件。有了once，应该用于在脚本执行期间同一个文件有可能被包含超过一次的情况下，想确保它只被包含一次以避免函数重定义，变量重新赋值等问题。
  2. isset() 函数：用于检测变量是否已设置并且非 NULL。
  3. $GET
    预定义的 $GET 变量用于收集来自 method=”get” 的表单中的值。
    从带有 GET 方法的表单发送的信息，对任何人都是可见的（会显示在浏览器的地址栏url）
  4. is_numeric() 函数用于检测变量是否为数字或数字字符串。
  5. exit() 函数输出一条消息，并退出当前脚本。
  6. php是弱类型语言，其变量比较包括松散比较和严格比较：
    松散比较：使用两个等号 == 比较，只比较值，不比较类型。
    严格比较：用三个等号 === 比较，除了比较值，也比较类型。
  7. intval() 函数用于获取变量的整数值。
    intval() 函数通过使用指定的进制 base 转换（默认是十进制），返回变量 var 的 integer 数值。
    但是！！！
    对于intval()而言，如果参数是字符串，则返回字符串中第一个不是数字的字符之前的数字串所代表的整数值。（如果字符串第一个是‘-‘，则从第二个开始算起。）
    有了这些就可以分析出，我们应该以get形式传入一个key=123
    得到了flag！
    flag{7430481c-683b-4534-95e7-dc6c06c66de5}

涉及的知识点

对出现的函数进行分析：

  1. strrev() 函数反转字符串。
    如：
  2. strlen() 函数返回字符串的长度
  3. substr()函数返回字符串的一部分。
    用法为：substr(string,start,length)
    其中：

string是待操作的字符串；
start规定在字符串的何处开始返回（必须有！）
正数 - 在字符串的指定位置开始
负数 - 在从字符串结尾开始的指定位置开始
0 - 在字符串中的第一个字符处开始
length规定被返回字符串的长度。默认是直到字符串的结尾。（可以不写）
正数 - 从 start 参数所在的位置返回的长度
负数 - 从字符串末端返回的长度

  1. ord() 函数返回字符串中第一个字符的 ASCII 值。
  2. chr() 函数从指定 ASCII 值返回字符。
  3. str_rot13() 函数对字符串执行 ROT13 编码，即把每一个字母在字母表中向前移动 13个字母。str_rot13()编码和解码都是由该函数完成，
    如果把已编码字符串作为参数，则返回原始字符串
